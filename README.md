# Telegram Бот для Реєстрації на Івент L'Oréal

Бот для автоматизованої реєстрації учасників розіграшу «Вгадай товар з L'Oréal» з інтеграцією Google Sheets.

## Функціонал

✅ **Автоматична реєстрація учасників** з збором:
- ПІБ
- Номер телефону
- Telegram username та ID
- Категорія відповіді (1 або 3)
- Відповідь учасника
- Автоматичний номерок

✅ **Захист від шахрайства:**
- Один користувач = одна відповідь на категорію
- Автоматична перевірка дублікатів по Telegram ID
- Обмеження часу роботи (понеділок-п'ятниця до 16:00)

✅ **Дві окремі Google таблиці:**
- Таблиця для категорії 1 (назва продукту + рекомендація)
- Таблиця для категорії 3 (назва продукту)

✅ **Автоматична нумерація:**
- Категорія 1, номер 1, 2, 3...
- Категорія 3, номер 1, 2, 3...

## Встановлення

### 1. Встановіть Python 3.8+

Перевірте версію:
```bash
python --version
```

### 2. Клонуйте/завантажте проект

```bash
cd c:\Users\Legion\Desktop\RegEvent
```

### 3. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 4. Створіть Telegram бота

1. Відкрийте Telegram і знайдіть [@BotFather](https://t.me/BotFather)
2. Відправте команду `/newbot`
3. Введіть назву бота (наприклад: "L'Oréal Event Bot")
4. Введіть username бота (має закінчуватися на `bot`, наприклад: `loreal_event_bot`)
5. Скопіюйте токен, який видасть BotFather

### 5. Налаштуйте Google Sheets

#### 5.1. Створіть Google Cloud проект

1. Перейдіть на [Google Cloud Console](https://console.cloud.google.com/)
2. Створіть новий проект
3. Увімкніть Google Sheets API та Google Drive API:
   - У меню зліва виберіть "APIs & Services" → "Library"
   - Знайдіть "Google Sheets API" та натисніть "Enable"
   - Знайдіть "Google Drive API" та натисніть "Enable"

#### 5.2. Створіть Service Account

1. Перейдіть у "APIs & Services" → "Credentials"
2. Натисніть "Create Credentials" → "Service Account"
3. Введіть назву (наприклад: "loreal-bot-service")
4. Натисніть "Create and Continue"
5. Надайте роль "Editor" → "Continue" → "Done"

#### 5.3. Створіть ключ

1. Знайдіть створений Service Account у списку
2. Натисніть на нього
3. Перейдіть на вкладку "Keys"
4. "Add Key" → "Create new key" → "JSON"
5. Файл `credentials.json` завантажиться автоматично
6. Перемістіть цей файл у папку проекту `c:\Users\Legion\Desktop\RegEvent\`

#### 5.4. Створіть Google таблиці

1. Відкрийте [Google Sheets](https://sheets.google.com/)
2. Створіть дві нові таблиці:
   - "L'Oréal Event - Категорія 1"
   - "L'Oréal Event - Категорія 3"

3. У кожній таблиці створіть лист з назвою **"Відповіді"**

4. Додайте заголовки в перший рядок (A1:H1):
   ```
   Номер | ПІБ | Телефон | Telegram Username | Telegram ID | Категорія | Відповідь | Дата і час
   ```

5. Скопіюйте ID кожної таблиці з URL:
   ```
   https://docs.google.com/spreadsheets/d/ВАШ_ID_ТАБЛИЦІ/edit
                                          ^^^^^^^^^^^^^^^^
   ```

6. Надайте доступ Service Account до таблиць:
   - Відкрийте файл `credentials.json`
   - Знайдіть поле `client_email` (наприклад: `loreal-bot-service@project.iam.gserviceaccount.com`)
   - У кожній Google таблиці натисніть "Share" (Поділитися)
   - Вставте email з `client_email`
   - Надайте права "Editor" (Редактор)
   - Зніміть галочку "Notify people" (не відправляти email)
   - Натисніть "Share"

### 6. Налаштуйте змінні оточення

1. Скопіюйте файл `.env.example` у `.env`:
```bash
copy .env.example .env
```

2. Відредагуйте файл `.env` та заповніть:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_від_BotFather
GOOGLE_SHEET_ID_CATEGORY_1=ID_таблиці_категорії_1
GOOGLE_SHEET_ID_CATEGORY_3=ID_таблиці_категорії_3
SHEET_NAME=Відповіді
GOOGLE_CREDENTIALS_FILE=credentials.json
```

## Запуск бота

```bash
python bot.py
```

Ви побачите повідомлення:
```
INFO - Успішно підключено до Google Sheets
INFO - Бот запущено!
```

## Використання

### Для учасників:

1. Знайдіть бота в Telegram за username
2. Натисніть `/start`
3. Слідуйте інструкціям бота:
   - Введіть ПІБ
   - Введіть номер телефону
   - Оберіть категорію (1 або 3)
   - Введіть відповідь

4. Отримайте підтвердження з номерком

### Обмеження:

- **Час роботи:** понеділок-п'ятниця до 16:00
- **Одна відповідь:** кожен користувач може надіслати лише одну відповідь в одній категорії
- **Можна брати участь у обох категоріях**

## Корисні команди

- `/start` - початок реєстрації
- `/cancel` - скасувати поточну реєстрацію

## Технічні деталі

### Перевірка дублікатів

Бот перевіряє Telegram ID користувача у відповідній таблиці. Якщо ID вже є, реєстрація блокується.

### Часова зона

Використовується часова зона `Europe/Kiev` (Київ, Україна).

### Логування

Всі дії бота логуються в консоль для моніторингу.

## Можливі проблеми та рішення

### Помилка "TELEGRAM_BOT_TOKEN не знайдено"
- Переконайтеся, що файл `.env` існує
- Перевірте, що токен правильно вказаний у `.env`

### Помилка підключення до Google Sheets
- Перевірте, що файл `credentials.json` існує у папці проекту
- Перевірте, що Service Account має доступ до таблиць (Rights: Editor)
- Перевірте, що ID таблиць правильні у `.env`

### Бот не відповідає
- Перевірте, що бот запущений (`python bot.py`)
- Перевірте інтернет-з'єднання
- Перегляньте логи у консолі

### Помилка "У п'ятницю прийом відповідей зупиняється о 16:00"
- Це нормальна поведінка бота
- Реєстрація відновиться в понеділок

## Підтримка

При виникненні проблем:
1. Перевірте логи у консолі
2. Переконайтеся, що всі налаштування правильні
3. Перезапустіть бота
